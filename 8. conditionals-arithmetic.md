## âš ï¸ 8. Common Pitfalls

| Mistake | Fix |
|---------|-----|
| Using variables without `"` | Always quote: `"$var"` |
| Using `[` instead of `[[` for logic | Use `[[` when possible |
| Forgetting spaces in `[ $a==5 ]` | Needs spacing: `[ "$a" == "5" ]` |
| Arithmetic in `[ $((a+1)) -eq 2 ]` | Use `(( ))` for clarity |
