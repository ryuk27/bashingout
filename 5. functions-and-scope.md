## ğŸ” Return Values

Functions in Bash can't *return* values like in other languages, but you can use:

1. `echo`:

```bash

  echo $(( $1 * $2 ))
}

result=$(multiply 4 5)
echo "Result: $result"
```

2. `return` (for exit status only â€” between 0 to 255):

```bash

  if (( $1 % 2 == 0 )); then
    return 0
  else
    return 1
  fi
}

is_even 4
if [ $? -eq 0 ]; then
  echo "Even"
else
  echo "Odd"
fi
```

## ğŸŒ€ Local vs Global Variables

By default, **variables are global** in Bash functions.

```bash
greet() {
  name="John"
}
greet
echo "$name" # Outputs: John
```

To **restrict variable to a function**, use `local`:

```bash
greet() {
  local name="Alice"
  echo "Hello, $name"
}
greet
echo "$name" # Will be empty
```

## ğŸ§ª Practical Example: Check File Size

```bash

  local file=$1
  if [ -f "$file" ]; then
    size=$(stat -c%s "$file")
    echo "Size of $file: $size bytes"
  else
    echo "$file does not exist."
  fi
}

check_file_size /etc/passwd
```
