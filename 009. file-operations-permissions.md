## ğŸ” 4. Permissions Management

| Command | Purpose |
|---------|---------|
| `chmod` | Change permission |
| `chown` | Change owner |
| `chgrp` | Change group |

ğŸ”¹ `chmod` Examples
```bash
chmod 755 script.sh # rwxr-xr-x
chmod +x script.sh # Add execute permission
chmod u+x file.txt # Add execute to user only
```

| Symbol | Meaning |
|--------|---------|
| `u` | user (owner) |
| `g` | group |
| `o` | others |
| `a` | all |

ğŸ”¹ `chown` Example
```bash

```
